Dein eigenes Second Brain

Du bekommst gleich deinen eigenen KI-Wissensmanager — ein externes Gedächtnis, das mitdenkt.

Er merkt sich, was dich beschäftigt. Er erkennt Zusammenhänge. Und er wächst mit dir über die Zeit.

Die Einrichtung dauert etwa 15 Minuten. Du brauchst nur einen Laptop.

Was ist das genau?

Dein Second Brain besteht aus zwei Teilen:

Obsidian

Ein Notizbuch auf deinem Computer. Hier schreibst du Gedanken, Reflexionen, Projektnotizen — alles was dich beschäftigt.

Dein KI-Begleiter

Eine KI, die dein Notizbuch kennt. Du redest mit ihr in VS-Code. Sie hilft dir Gedanken zu ordnen, erinnert dich an frühere Überlegungen und baut Verbindungen auf.

Du wählst einen Namen für deinen Begleiter — das macht ihn persönlicher.

VS-Code installieren

VS-Code ist ein kostenloser Code-Editor. Dein KI-Begleiter lebt dort.

  1. Öffne code.visualstudio.com
  2. Lade die Version für dein System herunter
  3. Installieren und öffnen

Falls du VS-Code schon hast — weiter zum nächsten Schritt.

Claude Code installieren

Claude Code ist die KI-Extension für VS-Code.

  1. Öffne VS-Code
  2. Drücke Ctrl+Shift+X (Extensions)
  3. Suche nach "Claude Code" (von Anthropic)
  4. Klicke auf Install

Claude-Zugang einrichten

Claude Code braucht einen Zugang zu Claude. Zwei Optionen:

Claude Max Abo (einfacher)

Wenn du ein Claude-Abo hast (Max-Plan, ab $100/Monat), kannst du dich damit anmelden. Beim Start auf "Login with Claude.ai" klicken.

API-Key (flexibler)

Auf console.anthropic.com einen Account erstellen, API-Key generieren, und beim Start eingeben. Kostet nach Verbrauch (~5–20 $/Monat).

Kurzer Test

Prüfe, ob alles läuft:

  1. In VS-Code: Ctrl+Shift+P
  2. Tippe "Claude Code: Open Chat"
  3. Oder einfach Ctrl+L
  4. Schreib "Hallo"

Wenn Claude antwortet, läuft alles.

Beim ersten Mal wirst du nach deinem Login gefragt — das ist normal.

Jetzt wird's eingerichtet

Ab hier übernimmt Claude Code die Arbeit. Du musst nur noch:

  1. Die Setup-Datei herunterladen
  2. Die Datei in VS-Code öffnen (Datei ziehen oder Datei → Öffnen)
  3. In Claude Code tippen: Setup my second brain

Claude Code sieht die Datei und macht den Rest. Er wird dir zwischendurch Fragen stellen — z.B. wie dein Begleiter heißen soll.

Fertig!

Dein Second Brain ist eingerichtet.

So nutzt du es ab jetzt

  • In Obsidian schreibst du Gedanken, Journal-Einträge, Projektnotizen
  • In Claude Code (VS-Code) redest du mit deinem Begleiter
  • Er hilft dir sortieren, erinnert dich an frühere Gedanken und baut Verbindungen auf

Fang an mit: Öffne den Steckbrief in Obsidian und stell dich vor. Das ist die Grundlage für alles Weitere.
