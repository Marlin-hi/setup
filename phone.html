Willkommen
Dauert ca. 5 Min

Mika aufs Handy

Dein Vault frisch von GitHub auf dein Handy klonen.| Danach synct alles automatisch.

Vorbereitung

Alten Vault entfernen

Falls du schon einen Mika-Vault auf dem Handy hast, der nicht mehr richtig synct:

  1. In Obsidian: lang auf den Vault-Namen drücken
  2. "Remove vault" wählen
  3. Nur aus der App entfernen — Dateien können bleiben

Kein alter Vault da? Dann einfach weiter.

Schritt 1

Leeren Vault erstellen

Obsidian braucht erstmal einen leeren Vault. Den füllen wir gleich mit dem Klon.

  1. Obsidian öffnen
  2. "Create new vault" tippen
  3. Name: mika
  4. Erstellen

Schritt 2

Git-Plugin installieren

Das Plugin verbindet Obsidian mit GitHub.

  1. Einstellungen öffnen (Zahnrad unten links)
  2. Community Plugins → Restricted Mode ausschalten
  3. Browse tippen
  4. Nach "Git" suchen
  5. Install → dann Enable

Schritt 3

GitHub Token erstellen

Obsidian Git braucht einen Token, um auf dein Repo zugreifen zu können.

  1. Öffne den Link unten — er führt direkt zur Token-Erstellung
  2. Ablaufdatum: No expiration
  3. Haken bei repo ist schon gesetzt
  4. Generate token klicken
  5. Den Token kopieren — er wird nur einmal angezeigt!
Token erstellen auf GitHub

Du musst bei GitHub angemeldet sein. Falls du schon einen Token hast, kannst du den wiederverwenden.

Schritt 4

Repo klonen

Zuerst Zugangsdaten eintragen, dann klonen.

  1. Einstellungen → Git (unter Community Plugins)
  2. Runterscrollen zu Authentication/commit author
  3. Username: Marlin-hi
  4. Password/Token: den Token von eben einfügen

Dann den Klon starten:

  1. Von oben nach unten wischen → Kommandopalette
  2. Tippe "Clone"
  3. Wähle "Git: Clone an existing remote repo"
  4. Repo-URL einfügen (kopiere sie hier):
https://github.com/Marlin-hi/mika.git
  1. Vault-Ordner: . (ein Punkt — klont direkt in den Vault)

Das Klonen kann einen Moment dauern. Danach startet Obsidian automatisch neu.

Schritt 5

Plugins aktivieren

Nach dem Klon fragt Obsidian, ob du den Plugins vertraust.

  1. "Trust author and enable plugins" wählen
  2. Falls das nicht kommt: Einstellungen → Community Plugins → Restricted Mode aus
  3. Prüfe dass Git und Mika Timestamps aktiv sind

Die Sync-Einstellungen kommen automatisch mit dem Repo. Du musst nichts konfigurieren.

Läuft

Mika ist auf deinem Handy. Ab jetzt synct der Vault automatisch mit GitHub.

Automatisch — Änderungen werden alle 5 Minuten gepusht

Beim Öffnen — Obsidian pullt automatisch die neuesten Änderungen

Timestamps — Das Plugin loggt wann du welche Notizen öffnest und bearbeitest
